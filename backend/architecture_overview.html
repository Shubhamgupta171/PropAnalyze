3. Technical Specifications

Component Technology
Backend Runtime Node.js
Database PostgreSQL (Raw SQL, pg driver)
Authentication JWT (JSON Web Tokens)
File Storage Cloudinary
Middleware Multer (Uploads), Helmet (Security)

4. Data Strategies

  • Raw SQL: Direct control over optimization and zero boilerplate.
  • JSONB: Used for flexible fields like property features and analysis inputs.
  • UUIDs: Used for all primary keys to ensure global uniqueness and security.
